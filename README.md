# vo-words - 单词学习应用

一款轻量级的网页单词学习工具，支持多种学习模式和智能复习。

## 功能特点

- **📝 拼写练习**：键盘输入拼写，实时验证
- **🔗 连线配对**：英汉连线，强化记忆
- **📇 翻翻卡**：3D翻转卡片，快速复习
- **🔊 自动发音**：使用有道词典API，标准发音
- **📊 学习统计**：记录学习进度，量化效果
- **🔄 智能复习**：基于艾宾浩斯遗忘曲线的间隔重复算法

## 快速开始

1. 双击 `index.html` 在浏览器中打开
2. 选择学习模式开始学习
3. 支持上传自定义词库

## 词库格式

应用支持以下词库格式：

### JSON 格式（推荐）
```json
[
  {
    "word": "people",
    "phonetic": "/ˈpiːpl/",
    "meaning": "人类，人们",
    "category": "人物类",
    "example": "People love music."
  }
]
```

### Markdown 格式
兼容 Obsidian 表格格式：
```markdown
| 词汇 | 音标 | 中文 |
|:---|:---|:---|
| people | - | 人类 |
| teacher | /ˈtiːtʃər/ | 老师 |
```

### CSV 格式
```csv
word,phonetic,meaning,category
people,,人类,人物类
teacher,/ˈtiːtʃər/,老师,人物类
```

## 学习模式说明

### 拼写练习
- 显示单词和音标
- 键盘输入拼写
- Enter 提交，正确自动下一词

### 连线配对
- 左侧5个英文单词
- 右侧对应的中文意思
- 点击进行连线配对

### 翻翻卡
- 点击卡片翻转查看答案
- 标记"认识"或"不认识"
- 不认识的词会优先复习

### 复习模式
- 根据遗忘曲线安排复习
- 首页显示今日待复习数量
- 答对升级，答错降级

## 数据存储

学习进度保存在浏览器 LocalStorage 中：
- 今日学习词数
- 累计学习词数
- 每个单词的熟练度
- 下次复习时间

**注意**：清除浏览器数据会丢失学习进度，建议定期导出。

## 开发说明

这是一个单页应用，所有代码内联在 `index.html` 中，便于分发和使用。

核心模块：
- `app` - 应用主控制器
- `app.storage` - LocalStorage 存储模块
- `app.parser` - 词库解析模块
- `app.spellingMode` - 拼写模式
- `app.matchingMode` - 连线模式
- `app.flashcardMode` - 翻翻卡模式
- `app.reviewMode` - 复习模式
- `app.statsMode` - 统计模块
